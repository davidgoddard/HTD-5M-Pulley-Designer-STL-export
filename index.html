Manufacturing note

This app generates the groove from tangential arc geometry using OD offset, groove depth, plunge radius, and floor/corner radius rather than a percentage-width slot. The lower flange is square because it prints from the bed without bridging; only the optional upper flange uses the print-friendly taper. It is still not a certified manufacturer profile unless those dimensions match the exact belt maker's published data and tolerances.
